variables -
rules for variable name :
- starting should be alphabet : 'a-z' & 'A-Z'.
- do not use number at starting of variable name.
- can only use _ underscore, should use in-between variable name, avoid using at start or end of variable name.
- meaningful
- python case sensitive :
    - 'a' and 'A' are different.
- variable names should not be a keywords.(if, else, while, etc)
